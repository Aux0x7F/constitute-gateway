# Documentation Index

This folder is the contributor-facing documentation surface for `constitute-gateway`.

## Contents
- `docs/PROTOCOL.md`: discovery and relay/transport protocol contracts
- `docs/OPERATIONS.md`: deployment and runtime hardening guidance
- `docs/DEVELOPMENT.md`: local workflow, quality gates, and release expectations

## Publishing
GitHub renders all markdown files in this folder directly.
A docs workflow can also publish generated Rust API docs to GitHub Pages.
